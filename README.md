# go-yolo
使用golang调用darknet接口

>备注: darknet官方,有python调用接口的脚本噢

### 快速开始
假设你已经完成了darknet的编译安装
1. 切换到darknet的工作目录,然后执行命令:
    ```
    cd darknet && git clone https://github.com/bzyy/go-yolo.git  && cd go-yolo
    ```

2. 执行: `go run detector.go`
